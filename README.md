# 📂 취미 커뮤니티 프로젝트

이 폴더는 취미별 통계 분석 및 웹사이트 프로젝트 파일들을 포함하고 있습니다.

## 📁 폴더 및 파일 구조

### 1. 🌐 [hobby-community/](hobby-community/)
**메인 프로젝트 폴더**입니다. 웹사이트의 모든 소스 코드가 들어있습니다.
- **index.html**: 웹사이트 메인 페이지 (통계, 커뮤니티, 출석부)
- **styles.css**: 웹사이트 디자인 및 스타일
- **script.js**: 기능 구현 (차트, 로컬 저장소 등)
- **README.md**: 상세한 프로젝트 설명서

### 2. 🐍 [hobby_statistics.py](hobby_statistics.py)
Python으로 작성된 통계 데이터 생성 및 시각화 스크립트입니다.
- 실행 시 통계 이미지를 생성하고 데이터를 출력합니다.

### 3. 🔗 [index.html](index.html) (현재 폴더)
GitHub에 배포된 웹사이트로 쉽게 이동할 수 있게 도와주는 **접속 링크 생성기**입니다.
- 실행 후 GitHub 아이디를 입력하면 내 사이트로 바로 접속할 수 있습니다.

---

## 🚀 빠른 시작 가이드

1. **웹사이트 소스 코드 확인**: `hobby-community` 폴더로 들어가세요.
2. **통계 데이터 생성**: `hobby_statistics.py`를 Python으로 실행하세요.
3. **배포된 사이트 접속**: 이 폴더의 `index.html`을 실행하세요.
