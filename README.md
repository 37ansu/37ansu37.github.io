# 🎨 취미 커뮤니티 웹사이트

취미를 가진 사람들이 통계를 확인하고 서로 소통할 수 있는 인터랙티브 커뮤니티 플랫폼입니다.

## 📊 프로젝트 개요

이 웹사이트는 7가지 취미(뜨개질, 독서, 필사, 다이어리 꾸미기, 헬스, 요가, 수영)를 가진 사람들의 통계를 시각화하고, 각 취미별로 커뮤니티 공간을 제공합니다.

### ✨ 주요 기능

- **📈 통계 대시보드**: 취미별 인원 통계를 차트(Chart.js)로 시각화
- **💬 커뮤니티 탭**: 7개 취미별 독립적인 토론 공간
- **✍️ 댓글 시스템**: 각 커뮤니티에서 자유롭게 의견 공유 (LocalStorage 저장)
- **📝 출석부**: 방문자 출석 체크 및 실시간 출석 현황 확인
- **👀 방문자 카운터**: 웹사이트 총 방문자 수 집계
- **🎨 밝은 테마**: 화사하고 깔끔한 라이트 모드 디자인 (Glassmorphism 적용)
- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원

## 📁 프로젝트 구조

```
hobby-community/
├── index.html          # 메인 HTML (통계, 커뮤니티, 출석부)
├── styles.css          # 스타일시트 (라이트 테마, 애니메이션)
├── script.js           # 로직 (차트, 댓글/출석 저장, 인터랙션)
├── hobby_data.json     # 통계 데이터
├── DEPLOYMENT.md       # GitHub 배포 가이드
├── GITHUB_LINK.md      # 배포된 링크 확인용
└── README.md           # 이 파일
```

## 🚀 시작하기

### 1️⃣ 웹사이트 실행

`hobby-community` 폴더의 `index.html` 파일을 브라우저에서 엽니다:

**방법 1: 파일 탐색기에서**
```
d:\_0 AHN SU HYUN\1. 대학원(23학번)\2. 정규학기\h.w\hobby-community\index.html
```
파일을 더블클릭하거나 우클릭 → "연결 프로그램" → 웹 브라우저 선택

**방법 2: 로컬 서버 사용 (권장)**
```bash
cd "d:\_0 AHN SU HYUN\1. 대학원(23학번)\2. 정규학기\h.w\hobby-community"
python -m http.server 8000
```
그 다음 브라우저에서 `http://localhost:8000` 접속

## 🛠️ 기술 스택

### Backend / Data
- **Python 3.x**: 통계 생성
- **LocalStorage**: 브라우저 내 데이터 저장 (댓글, 출석, 방문자 수)

### Frontend
- **HTML5**: 시맨틱 웹 구조
- **CSS3**: CSS Variables, Flexbox/Grid, Animations
- **JavaScript (ES6+)**: DOM 조작, 이벤트 처리
- **Chart.js**: 데이터 시각화 라이브러리
