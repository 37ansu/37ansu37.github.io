# 🌐 내 웹사이트 접속 링크

GitHub에 코드를 업로드하고 Pages 설정을 마치셨다면, 아래 링크로 접속할 수 있습니다.

## 👉 접속 주소
(본인의 GitHub 아이디를 넣어주세요)

```
https://[본인아이디].github.io/hobby-community/
```

예시) 만약 아이디가 `gildong`이라면: `https://gildong.github.io/hobby-community/`

---

## ❓ 접속이 안 되나요?

1. **코드를 올리지 않으셨나요?**
   - `DEPLOYMENT.md` 파일을 보고 코드를 GitHub에 올려주세요.

2. **설정을 아직 안 하셨나요?**
   - GitHub 저장소의 `Settings` > `Pages` 메뉴로 이동합니다.
   - **Branch**를 `main`으로 선택하고 `Save`를 누르세요.

3. **방금 설정을 마치셨나요?**
   - 설정 후 웹사이트가 보일 때까지 **약 1~2분** 정도 걸릴 수 있습니다. 잠시 기다렸다가 새로고침 해보세요!
